# GlobalMart Data & AI Ecosystem

Simulation of a modular data and AI ecosystem for a fictional global e-commerce company.

## Objective

Build an end-to-end data and AI project covering synthetic data generation, data cleaning, data modeling, business analysis, visualization, automation, and AI-powered analytical workers.

The goal is to simulate how a real company could use data and AI to support decision-making, detect problems, generate insights, and automate repetitive analytical tasks.

## Project Structure

* `data/` → raw and processed datasets
* `notebooks/` → exploratory analysis and experiments
* `src/` → data generation, cleaning, and processing scripts
* `sql/` → database modeling and analytical queries
* `reports/` → business reports and insights
* `dashboards/` → Power BI dashboards
* `ai_workers/` → AI-powered analytical assistants

## AI Workers

This project will include specialized AI workers built on top of the GlobalMart ecosystem.

Planned workers:

* `csv_analyst/` → analyzes CSV files and generates KPIs, insights, charts, and summaries
* `executive_summarizer/` → transforms metrics into executive reports
* `sql_chat/` → helps generate and explain SQL queries
* `anomaly_detector/` → identifies unusual orders, outliers, and possible operational issues
* `forecasting_assistant/` → supports demand and revenue forecasting

## Technologies

* Pandas

## Current Status

This project is currently in the initial development phase.

The first focus is building the synthetic data foundation and creating the first AI worker: a CSV Analyst.

## Note

The data used in this project is synthetic and generated to simulate a realistic e-commerce environment, including seasonality, missing values, outliers, and business inconsistencies.
