# GlobalMart Data Ecosystem

Simulation of a data ecosystem for a fictional global e-commerce company.

## Objective

Build an end-to-end data analysis project, covering synthetic data generation, data cleaning, modeling, analysis, and visualization.

## Project Structure

* `data/` → raw and processed data
* `notebooks/` → Python analysis notebooks
* `src/` → data generation and processing scripts
* `reports/` → reports and insights

## Technologies

* Python (Pandas, NumPy, Faker)
* SQL
* Excel
* Power BI

## Note

The data used in this project is synthetic and was generated to simulate a realistic e-commerce environment.
